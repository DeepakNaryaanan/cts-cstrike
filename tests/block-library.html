CrowdStrike Block Library

This library documents all custom EDS blocks built for the CrowdStrike homepage migration. Each block section below contains a live rendered example using real CrowdStrike content. Copy the authored structure into da.live or Google Docs to use the block on any page.

Blocks covered: cs-hero • cs-resource-cards • cs-feature-pillars • cs-analyst-recognition • cs-pricing-table • cs-testimonials • cs-product-showcase • cs-adversary-cta

cs-hero — Hero Carousel

Full-width hero carousel with rotating panels. Each row in the block table represents one slide. The block auto-advances and supports dot and arrow navigation. An empty row separates individual panels.

Variants: default (carousel) | static | dark | image

Default variant — rotating carousel (2 panels)

Author by repeating the 5-row panel pattern (eyebrow, headline, body, CTA row, background image). Insert a fully empty row between panels as a separator.

cs-resource-cards — Resource and News Cards

Horizontal strip of resource and news cards. Renders as a 3-column CSS Grid on desktop (900px+), 2 columns at 600px, single column on mobile. Each card is a self-contained group of rows: optional image, tag/category label, headline link, and optional teaser text.

Variants: default | featured | no-image

No-image variant — text-only cards

Add no-image to the block name. Image cells are omitted entirely. Each card is 3 rows: tag label, headline link, optional teaser. This is the variant used on the CrowdStrike homepage.

cs-pricing-table — Pricing Plan Comparison

Tabbed pricing section with plan comparison. Supports monthly and annual billing tabs. Plans are authored as columns; features are rows. The recommended keyword in the recommended-flag row activates a visual badge on that plan.

Variants: default (tabbed) | single-billing | compact | enterprise

Default variant — monthly/annual tabs, 3 plans

Row 1: section heading + subtitle (2 columns). Row 2: billing tab labels (2 columns). Row 3: plan names (1 column per plan). Row 4: monthly prices. Row 5: annual prices. Row 6: CTA links. Row 7: recommended flag (empty = not recommended, literal text "recommended" = flagged). Rows 8+: feature rows using format Feature name: included or Feature name: not included.

cs-adversary-cta — Adversary CTA

Full-width dark call-to-action section. Minimal content: optional eyebrow label, headline, body text, primary CTA button, optional secondary CTA. The eyebrow cell is always row 1 — leave it empty if not needed.

Variants: default | split | light | with-eyebrow

Default variant — centred, dark background

5 rows: eyebrow (empty = no eyebrow rendered), headline, body text, primary CTA link, secondary CTA (empty = not rendered). No JavaScript required for this variant — CSS-only block.
